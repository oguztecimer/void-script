Create `editor-ui/src/components/BreadcrumbBar.tsx`:

1. **Pure utility function** (top of file, not exported):
```typescript
const DEF_RE = /^\s*def\s+([a-zA-Z_]\w*)\s*\(/;

function findEnclosingFunction(content: string, cursorLine: number): string | null {
  if (!content || cursorLine < 1) return null;
  const lines = content.split('\n');
  const startIdx = Math.min(cursorLine - 1, lines.length - 1);
  for (let i = startIdx; i >= 0; i--) {
    const m = DEF_RE.exec(lines[i]);
    if (m) return m[1];
  }
  return null;
}
```

2. **Component** using Zustand selectors:
- `cursorLine` via `useStore((s) => s.cursorLine)` -- primitive, re-renders only on change.
- `activeTabContent` via `useStore((s) => { const t = s.tabs.find(tab => tab.scriptId === s.activeTabId); return t?.content ?? ''; })` -- returns string (primitive equality).
- `activeTabName` via `useStore((s) => { const t = s.tabs.find(tab => tab.scriptId === s.activeTabId); return t?.name ?? null; })` -- returns string|null (primitive equality).
- Derive `fnName` via `useMemo(() => findEnclosingFunction(activeTabContent, cursorLine), [activeTabContent, cursorLine])`.
- If `!activeTabName` return `null` (no active tab = no breadcrumb).
- Render: a `.bar` div containing:
  - `<span className={styles.segment}>{activeTabName}.vs</span>` (filename with .vs extension)
  - When `fnName` is non-null: `<span className={styles.chevron}> > </span>` followed by `<span className={styles.segmentActive}>{fnName}</span>`

3. **Breadcrumb is display-only** -- no click handlers, no navigation. Non-interactive.

Create `editor-ui/src/components/BreadcrumbBar.module.css`:
- `.bar`: `display: flex; align-items: center; height: 24px; flex-shrink: 0; background-color: var(--bg-panel); border-bottom: 1px solid var(--border-default); padding: 0 12px; font-size: 12px; font-family: var(--font-ui);`
- `.segment`: `color: var(--text-secondary); padding: 0 2px;`
- `.segmentActive`: `color: var(--text-primary); padding: 0 2px;`
- `.chevron`: `color: var(--text-secondary); margin: 0 2px;`

Key: `flex-shrink: 0` on `.bar` is CRITICAL -- without it the breadcrumb bar collapses inside the flex column layout (Phase 6 lesson, StatusBar pattern).
